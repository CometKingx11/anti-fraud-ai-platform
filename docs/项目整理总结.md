## 🎯 使用指南

### 快速开始

```bash
# 1. 查看项目结构
cat docs/PROJECT_STRUCTURE.md

# 2. 查看脚本使用说明
cat scripts/README.md

# 3. 查看测试说明
cat tests/README.md

# 4. 启动应用
python run.py
```

---

### 添加新功能

```
1. 在 app/ 目录下添加代码
   - models/     # 数据模型
   - views/      # 视图控制
   - services/   # 业务逻辑
   - templates/  # HTML 模板

2. 在 tests/ 目录下添加测试
   - test_new_feature.py

3. 在 docs/ 目录下添加文档
   - features/新功能说明.md
```

---

### 运行测试

```bash
# 运行所有测试
python -m pytest tests/ -v

# 运行单个测试文件
python -m pytest tests/test_models.py -v

# 生成测试报告
python -m pytest tests/ --cov=app --cov-report=html
```

---

### 执行脚本

```bash
# 初始化数据库
python scripts/init_db.py

# 迁移数据库
python scripts/migrate_db.py

# 清空数据库（危险！）
python scripts/clean_db.py
```

## ⚠️ 注意事项

### 1. 不要提交到 Git 的文件

```
# 环境变量
.env

# 数据库
instance/*.db

# 上传文件
uploads/*

# Python 缓存
__pycache__/
*.pyc
```

---

### 2. 备份重要数据

```bash
# 备份数据库
cp instance/anti_fraud_dev.db instance/anti_fraud_dev.db.backup

# 备份上传文件
tar -czf uploads_backup.tar.gz uploads/
```

---

### 3. 使用虚拟环境

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```
