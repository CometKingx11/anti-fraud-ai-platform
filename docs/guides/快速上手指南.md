# 🚀 快速上手指南

## 📋 功能概述

本次更新实现了两大核心功能：

### 1. 🔗 风险链接智能检测
- **自动识别**学生提交的可疑链接
- **三重检测**：360 API → 腾讯 API → AI 大模型
- **智能评分**：发现风险链接自动加分（+10~20 分）

### 2. 📝 动态问卷管理
- **完全可配置**：管理员可自定义题目、评分规则
- **即时生效**：修改后学生端立即更新
- **灵活扩展**：支持添加新题、调整权重

## ⚡ 5 分钟快速开始

### 步骤 1：安装依赖

```bash
pip install Flask-WTF==1.2.1
```

或更新所有依赖：
```bash
pip install -r requirements.txt
```

### 步骤 2：配置环境变量

复制示例配置文件：
```bash
cp .env.example .env
```

编辑 `.env` 文件，填入你的 API Key：
```bash
SECRET_KEY=your-secret-key-here-at-least-32-bytes-long
DASHSCOPE_API_KEY=your-dashscope-api-key
360_API_KEY=your-360-api-key  # 可选
TENCENT_API_KEY=your-tencent-api-key  # 可选
```

### 步骤 3：初始化数据库

```bash
python init_questionnaire_db.py
```

输出：
```
✅ 成功初始化 30 道问卷题目
   - 认知维度：10 题
   - 行为维度：10 题
   - 经历维度：8 题
   - 开放题：2 题
```

### 步骤 4：启动应用

```bash
python run.py
```

访问：http://localhost:5000

## 📊 评分规则说明

### 基础分计算
```
总分 = 认知维度 (40%) + 行为维度 (40%) + 经历维度 (20%)

认知维度：第 1-10 题，正向计分（分数越高越安全）
行为维度：第 11-20 题，反向计分（分数越高风险越大）
经历维度：第 21-28 题，正向计分
```

### URL 风险加分
```
发现 1 个风险链接：+10 分
发现 2 个风险链接：+20 分（封顶）
发现 3 个及以上：+20 分

最终得分 = 基础分 + URL 风险分
```

### 风险等级
| 分数范围 | 风险等级 | 颜色标识 |
|---------|---------|---------|
| 0-30 分 | 低风险 | 🟢 绿色 |
| 31-55 分 | 中风险 | 🟡 黄色 |
| 56-80 分 | 高风险 | 🟠 橙色 |
| 81-120 分 | 极高风险 | 🔴 红色 |

## 🔧 常见问题

### Q1: 没有 360/Tencent API Key 能用吗？

**A:** 可以！系统会自动降级到 AI 大模型检测。

```
360 API (失败) → 腾讯 API (失败) → AI 分析 (成功) ✅
```

建议：开发阶段可以使用 AI 兜底，生产环境建议配置真实 API。

### Q2: 如何获取 360 API Key？

**A:** 
1. 访问 https://openapi.360.cn/
2. 注册开发者账号
3. 创建应用，申请"网址安全检测"API
4. 审核通过后获取 API Key
5. 免费额度：每日 1000 次

### Q3: 问卷题目能恢复默认吗？

**A:** 可以，重新运行初始化脚本：
```bash
# 先清空数据库（可选）
python clean_db.py

# 重新初始化
python init_questionnaire_db.py
```

### Q4: URL 检测会影响性能吗？

**A:** 
- 单个 URL 检测：约 0.5-2 秒
- 批量检测：并行处理，影响不大
- 建议：生产环境开启本地缓存
