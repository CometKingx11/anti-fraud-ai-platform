## 📋 详细操作步骤

### 场景 1：创建单个学生账户

1. 访问 http://localhost:5000/admin/users/create
2. 填写信息：
   ```
   学号：20240001
   姓名：张三
   邮箱：zhangsan@example.com
   角色：学生
   密码：student123
   ```
3. 点击"创建用户"
4. ✅ 完成！系统会自动跳转到用户列表

### 场景 2：批量导入 100 个学生

1. 准备 CSV 文件（Excel 另存为 CSV UTF-8 格式）
   ```csv
   student_id,name,email,role,password
   20240001，张三，zhangsan@example.com,student,123456
   20240002，李四，lisi@example.com,student,123456
   ...（更多行）
   ```

2. 访问 http://localhost:5000/admin/users/import
3. 点击"下载示例模板"参考格式
4. 选择你的 CSV 文件
5. 点击"开始导入"
6. ✅ 完成！系统会显示导入结果

### 场景 3：处理忘记密码

1. 学生联系管理员："老师，我忘记密码了"
2. 管理员访问用户列表：http://localhost:5000/admin/users
3. 找到该学生，点击 🔑 钥匙图标
4. 确认操作
5. 系统显示新密码：`aB3dE7fG`
6. 将新密码告诉学生
7. ✅ 完成！学生可以用新密码登录

### 场景 4：禁用违规用户

1. 发现某用户存在违规行为
2. 访问用户列表
3. 找到该用户
4. 点击"禁用"按钮
5. 确认禁用
6. ✅ 完成！该用户将无法再次登录

## ⚠️ 重要提示

### 安全提醒
1. ⚠️ **不要删除自己** - 系统会阻止删除当前登录的管理员
2. ⚠️ **谨慎删除用户** - 删除后数据无法恢复
3. ⚠️ **保护好管理员账号** - 拥有所有权限
4. ⚠️ **定期备份数据库** - 防止数据丢失

### 使用建议
1. ✅ 首次登录后修改默认密码
2. ✅ 定期审查用户列表
3. ✅ 及时禁用毕业/离职用户
4. ✅ 批量导入前先测试小批量

## 🔧 技术细节（开发者查看）

### API 端点
```python
GET  /admin/users                      # 用户列表
GET  /admin/users/create               # 创建用户页面
POST /admin/users/create               # 执行创建
GET  /admin/users/{id}/edit            # 编辑用户页面
POST /admin/users/{id}/edit            # 执行编辑
POST /admin/users/{id}/delete          # 删除用户
POST /admin/users/{id}/reset-password  # 重置密码
POST /admin/users/{id}/toggle-status   # 切换状态
GET  /admin/users/import               # 批量导入页面
POST /admin/users/import               # 执行导入
GET  /admin/users/download-template    # 下载模板
```

### 数据库模型
```python
class User(db.Model):
    id              # 主键
    student_id      # 学号（唯一）
    name            # 姓名
    email           # 邮箱
    password_hash   # 密码哈希
    role            # 角色
    is_active       # 是否启用
    last_login      # 最后登录
    created_at      # 创建时间
    updated_at      # 更新时间
```

## 📞 遇到问题？

### 常见问题解答

**Q: 访问 /admin/users 显示"权限不足"**
A: 请确保使用管理员账号（12345678）登录

**Q: 创建用户时提示"学号格式不正确"**
A: 学号必须是 8-12 位纯数字，例如：20240001

**Q: 批量导入失败**
A: 检查 CSV 文件编码是否为 UTF-8，第一行是否是表头

**Q: 忘记密码怎么办**
A: 如果是管理员忘记密码，需要直接修改数据库；普通用户可联系管理员重置

**Q: 如何添加教师角色**
A: 创建或编辑用户时，角色选择"教师"即可
