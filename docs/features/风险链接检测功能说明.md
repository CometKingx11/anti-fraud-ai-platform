### 1.2 评分标准调整

#### 原有评分规则
```
基础分 = 认知维度 (40%) + 行为维度 (40%) + 经历维度 (20%)
总分范围：0-100 分
```

#### 新增评分规则
```
最终得分 = 基础分 + URL 风险加分

URL 风险加分规则：
- 发现 1 个风险链接：+10 分
- 发现 2 个风险链接：+20 分
- 发现 3 个及以上：+20 分（封顶）
```

#### 风险等级划分
```
0-30 分    → 低风险
31-55 分   → 中风险
56-80 分   → 高风险
81 分以上  → 极高风险
```

**示例：**
- 学生基础分 65 分（高风险）
- 提交的可疑链接被 360 API 识别为诈骗网站
- 最终得分 = 65 + 10 = 75 分（仍为高风险）

## 🔍 技术实现细节

### 4.1 URL 提取算法

```python
import re

def extract_urls_from_text(text: str) -> List[str]:
    """从文本中提取所有 URL"""
    url_pattern = r'https?://[^\s<>"{}|\\^`\[\]]+'
    urls = re.findall(url_pattern, text)
    
    # 标准化处理（www.开头添加 http://）
    normalized = []
    for url in urls:
        if url.startswith('www.'):
            url = 'http://' + url
        normalized.append(url)
    
    return list(set(normalized))  # 去重
```

### 4.3 动态评分计算

```python
# questionnaire.py 模型方法

@classmethod
def calculate_dimension_score(cls, answers: dict, dimension: str) -> int:
    """根据数据库配置计算维度得分"""
    questions = cls.query.filter_by(
        dimension=dimension,
        is_active=True
    ).all()
    
    total_score = 0
    for q in questions:
        answer_value = answers[f'q{q.question_number}']
        
        # 正向计分（认知/经历）
        if q.category != 'behavior':
            score = answer_value
        # 反向计分（行为风险）
        else:
            score = (q.max_score + q.min_score + 1) - answer_value
        
        total_score += score * q.weight
    
    return round(total_score)
```
