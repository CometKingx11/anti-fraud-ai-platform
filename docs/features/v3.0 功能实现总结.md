### 3. 🎯 评分标准升级

#### 原有规则
```
基础分 = 认知 (40%) + 行为 (40%) + 经历 (20%)
范围：0-100 分
```

#### 新规则
```
总分 = 基础分 + URL 风险分

URL 风险分:
- 0 个风险链接：0 分
- 1 个风险链接：+10 分
- 2 个及以上：+20 分

总分范围：0-120 分
```

#### 风险等级
```
0-30 分    → 低风险（绿色）
31-55 分   → 中风险（黄色）
56-80 分   → 高风险（橙色）
81-120 分  → 极高风险（红色）
```

## 📋 功能演示

### 场景 1：学生提交风险链接

**学生填写：**
> "收到短信说我银行卡涉嫌洗钱，让我点击 http://fake-bank.com/verify 验证身份"

**系统处理：**
1. 提取 URL：`http://fake-bank.com/verify`
2. 调用 360 API 检测
3. 识别为"钓鱼网站"
4. 基础分 55 分 + URL 风险分 10 分 = **65 分（高风险）**

**检测报告：**
```
⚠️ 检测到可疑链接
链接地址：http://fake-bank.com/verify
风险类型：钓鱼网站
判断来源：360 安全大脑
建议：不要点击陌生链接，立即删除短信
```

### 场景 2：管理员调整题目

**操作路径：**
管理员后台 → 问卷管理 → 编辑题目

**可修改内容：**
- 题目文字："你清楚电信网络诈骗的常见手段吗？"
- 分值范围：1-5 分 → 1-10 分
- 权重系数：1.0 → 1.5
- 选项文本：自定义每个选项的描述

**即时生效：**
修改后学生端问卷立即更新
