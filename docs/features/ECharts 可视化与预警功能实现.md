## 💡 自定义配置

### 修改图表颜色
在 `report.html` 的 JavaScript 中找到：
```javascript
itemStyle: {
    color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
        { offset: 0, color: '#0d6efd' },  // 起始颜色
        { offset: 1, color: '#0dcaf0' }   // 结束颜色
    ])
}
```

### 修改预警阈值
找到 `checkWarning` 函数：
```javascript
const isHighRisk = (riskLevel === '高风险' || riskLevel === '极高风险');
```

如需在中风险时就预警，改为：
```javascript
const isHighRisk = (riskLevel === '中风险' || riskLevel === '高风险' || riskLevel === '极高风险');
```

### 修改弹窗文案
在 HTML 中找到对应部分修改即可：
- 标题：`<h2 class="text-danger mb-3">高风险预警</h2>`
- 建议文本：`<p class="mb-4"><strong>建议：</strong>...</p>`
