# 🚀 Anti-Fraud AI Platform

基于大模型的多模态反诈风险智能评估系统

---

## 🧠 项目简介

本项目面向大学生反诈场景，构建一个融合 **大模型（LLM）+ 多模态数据分析 + 风险建模** 的智能评估平台，实现用户风险识别与个性化防诈建议生成。

---

## ✨ 核心功能

- 📊 问卷风险评估（认知 / 行为 / 经历）
- 🤖 大模型智能分析（通义千问）
- 🧾 自动生成风险评估报告（PDF）
- 🔍 URL 安全检测（多API + AI fallback）
- 🖼️ 图片 OCR 解析（提取诈骗关键词）
- 📈 数据可视化（ECharts）

---

## 🧠 技术亮点

- 多模态数据融合（文本 + 图片 + 链接）
- 大模型与规则模型融合（Hybrid Risk Scoring）
- Prompt Engineering 实现结构化分析
- 模块化 Flask 架构设计
- 支持扩展 Agent 系统能力
